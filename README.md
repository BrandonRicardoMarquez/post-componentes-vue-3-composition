# Ejemplo curso Vue 3

- [Preview page](https://postfavorit.netlify.app/)

Este es un ejemplo del curso vue.js 3, trabajando con componentes y el composition api.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
